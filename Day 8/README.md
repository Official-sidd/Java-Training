# 📦 Day 8 – Spring Core & JDBC

## 📖 Overview

Day 8 focuses on learning **Spring Core fundamentals** and **JDBC database connectivity**.

This day contains two modules:

1. **Spring Core Sample Application**
2. **JDBC Database Demo**

The objective is to understand how Spring manages objects using **Beans and Dependency Injection**, and how Java applications interact with relational databases using **JDBC**.

---

## 🧠 Concepts Covered

### 🔹 Spring Core
- Spring Beans
- Bean Scopes
- Dependency Injection (DI)
- Java Configuration
- XML Configuration
- Component Scanning
- Basic Unit Testing

### 🔹 JDBC
- JDBC Architecture
- Database Connection
- Executing SQL Queries
- PreparedStatement
- ResultSet Processing
